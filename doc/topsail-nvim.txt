1. topsail.nvim                                    *topsail-nvim-topsail.nvim*

A Neovim plugin for managing Kubernetes resources directly from your editor.


FEATURES                                  *topsail-nvim-topsail.nvim-features*

- Automatic detection of Kubernetes YAML files
- Apply and Create resources directly from Neovim
- Async operations that donâ€™t block the editor
- Configurable keymaps and notifications


INSTALLATION                          *topsail-nvim-topsail.nvim-installation*

Using lazy.nvim <https://github.com/folke/lazy.nvim>:

>lua
    {
        "yourusername/topsail.nvim",
        config = function()
            require("topsail").setup({
                -- your configuration here
            })
        end,
        dependencies = {
            -- Add any dependencies here
        }
    }
<


SETUP                                        *topsail-nvim-topsail.nvim-setup*

You can setup the plugin with custom configuration:

>lua
    require('topsail').setup({
        notify = true,  -- Enable notifications
        keymaps = {
            apply = '<leader>ka',
            create = '<leader>kc'
        }
    })
<


COMMANDS                                  *topsail-nvim-topsail.nvim-commands*

- `:KubernetesApply` - Apply the current YAML file as a Kubernetes resource.
- `:KubernetesCreate` - Create a new Kubernetes resource from the current YAML file.


MAPPINGS                                  *topsail-nvim-topsail.nvim-mappings*

Default mappings (can be customized in setup):

- `<leader>ka` - Apply the current Kubernetes resource
- `<leader>kc` - Create a new Kubernetes resource


CONFIGURATION                        *topsail-nvim-topsail.nvim-configuration*

Configuration options:

>lua
    {
        notify = true,      -- Enable/disable notifications
        keymaps = {         -- Customize keymaps
            apply = '<leader>ka',
            create = '<leader>kc'
        }
    }
<


REQUIREMENTS                          *topsail-nvim-topsail.nvim-requirements*

- Neovim >= 0.8.0
- kubectl installed and in your PATH
